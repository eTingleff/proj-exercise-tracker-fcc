Exercise tracker

Create a New User

Copy the ID in the resulting response for future queries

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example 1 (Get all of the user's logs):

/api/exercise/log?userId=5b9f4015d2505548c64b737a

Example 2 (Get all of the user's logs after a specific date):

/api/exercise/log?userId=5b9f4015d2505548c64b737a&from=2010-01-01

Example 3 (Get all of the user's logs between two dates):

/api/exercise/log?userId=5b9f4015d2505548c64b737a&from=2010-01-01&to=2050-01-01

Example 4 (Get all of the user's logs between two dates and limited):

/api/exercise/log?userId=5b9f4015d2505548c64b737a&from=2000-01-01&to=2050-01-01&limit=5
